# Бот «Напоминание о воде»

Telegram-бот на Python (telebot), который напоминает пить воду.

## Возможности
- Настройка частоты напоминаний
- Подсчёт выпитой воды за день
- Напоминание, сколько осталось выпить

## Технологии
- Python
- pyTelegramBotAPI

## Как запустить
```bash
pip install pyTelegramBotAPI
python bot.py
